# Simple CLI App in Golang

Search IPs and Names of Internet Servers by url.

### Install

run 

```
  go go build
```
the, to receive server name

```
  ./simple-cli-app server --host <amazon.com.br>
```
or, ti receive ip 

```
  ./simple-cli-app ip --host <amazon.com.br>
```
